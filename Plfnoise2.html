Plfnoise2 Generates quadratically interpolated random values at a rate.



superclass: Pattern



Plfnoise2(nSamps, lo, hi, length)

nSamps number of samples between new random values

lo, hi lower and upper boundary of values (default: 0.0 and 1.0)

length number of values produced  (default: inf)


// example



(

var a;

a = Plfnoise2(20, 0.0, 1.0, inf);

c = a.asStream.nextN(500);

w = GUI.window.new("Plfnoise2", Rect(10, 10, 540, 800));

// plot the values

c.plot(bounds: Rect(10, 10, 520, 380), discrete: true, parent: w);

)


(

var a;

a = Plfnoise2(20, 0.0, 1.0, inf);

a.asStream.nextN(1000).plot;

)
